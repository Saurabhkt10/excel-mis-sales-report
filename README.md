# Sales & Inventory MIS Report (Excel)

## Objective
To build an end-to-end MIS report using transaction data and master data mapping for management reporting.

## Tools Used
- Microsoft Excel
- Excel Tables
- VLOOKUP / XLOOKUP
- Pivot Tables

## Sheets Included
- Transactions: Raw transaction-level data
- Item_Master: Item code, category, and rate mapping
- MIS: Live pivot-based reports

## Key Reports
- Total Sales Summary
- Item-wise Sales
- Category-wise Sales
- Date-wise / Monthly Sales

## How to Use
Update transaction data and refresh pivot tables to update MIS reports.
